#
Este é um projeto de portfólio profissional desenvolvido para o desenvolvedor front-end **Bruno Henrique**. O objetivo foi criar uma interface moderna, minimalista e altamente performática utilizando apenas tecnologias puras (Vanilla), seguindo os melhores padrões de mercado.

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e acessível.
- **CSS3**: Estilização avançada com Variáveis Globais (Root), Flexbox, Grid e Animações.
- **JavaScript (ES6+)**: Lógica de interação, animações de scroll, player de música persistente e carregamento dinâmico.

## 📁 Estrutura de Pastas

```text
/
├── index.html          # Página Inicial (Hero & Skills)
├── sobre.html          # Sobre a trajetória
├── carreira.html       # Timeline profissional/acadêmica
├── projetos.html       # Galeria de projetos
├── contato.html        # Formulário de contato e redes sociais
├── src/
│   ├── css/
│   │   ├── root.css        # Variáveis globais e reset
│   │   ├── index.css       # Estilos da Home, Header e Footer
│   │   ├── sobre.css       # Estilos específicos da página Sobre
│   │   ├── carreira.css    # Estilos específicos da página Carreira
│   │   ├── projetos.css    # Estilos específicos da página Projetos
│   │   └── contato.css     # Estilos específicos da página Contato
│   ├── js/
│   │   └── main.js         # Lógica global (Menu, Scroll, Player, Loading)
│   ├── img/                # Imagens e assets visuais
│   └── audio/              # Arquivos de áudio para o player
│      └── docs/            # Aquvios .pdf  
└── README.md           # Documentação do projeto
```

## ✨ Funcionalidades Implementadas

- **Design Responsivo**: Adaptado para dispositivos móveis, tablets e desktops.
- **Menu Moderno**: Navegação fixa com efeito blur e menu hambúrguer para dispositivos móveis.
- **Animações de Scroll**: Elementos surgem suavemente conforme o usuário navega pela página.
- **Player de Música Persistente**: Player fixo que mantém a reprodução contínua mesmo ao navegar entre diferentes páginas do site (utilizando `localStorage`).
- **Loading Overlay**: Transição suave entre páginas com uma animação de carregamento moderna.
- **Microinterações**: Efeitos de hover em botões, imagens e cards para uma experiência mais fluida.
- **Formulário de Contato**: Interface funcional com validação visual e simulação de envio.

## 🛠️ Como Rodar o Projeto

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Para uma experiência completa de áudio (devido às políticas de segurança de navegadores), recomenda-se abrir o projeto através de um servidor local (ex: Extensão *Live Server* do VS Code).

Demonstrar competência técnica em desenvolvimento front-end, organização de código, design visual e experiência do usuário, servindo como a vitrine principal da carreira de Bruno Henrique.
